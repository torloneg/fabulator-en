# set

set lets you create/modify a variable.

**params**

| name | type | description | allow |
| :--- | :--- | :--- | :--- |
| var | code | allows you to set the value of a variable with javascript code |  |

## example:

```text
{{ username }}
{% set username = "joe" %}
{{ username }}
```

## output:

```text
jow
```
